# 15-Block-Puzzle-Solver
A program that solves the 15-Block Puzzle using A* Search. The 15-Block puzzle is when you have a 4x4 grid with 15 tiles labeled 1-15. The goal is to rearrange the tiles such that the tiles are organized numerically, with the open tile being on the bottom right of the grid. The program runs by taking an argument, which is a file containing a 4x4 puzzle.
